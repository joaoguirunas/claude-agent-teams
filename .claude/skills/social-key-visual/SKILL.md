# Social Key Visual — Identidade Visual de Campanha

## O que é um Key Visual

O KV é o elemento visual central de uma campanha — a "impressão digital" que deve ser reconhecível em qualquer peça, qualquer formato.

## Componentes de um KV

```
Key Visual =
  Imagem/elemento principal (hero)
  + Paleta de cores da campanha
  + Tipografia escolhida
  + Tom visual (mood)
  + Tratamento de marca (logo placement, safe zone)
```

## Processo de criação

### 1. Briefing de KV
```markdown
## KV Brief — [Campanha]

**Marca:** ...
**Campanha:** ...
**Plataformas:** ...
**Tom visual:** [energético|minimalista|editorial|cinematic|bold]
**Paleta de cores:** [base da marca] + [cores de campanha]
**Tipografia:** [fonte principal] + [fonte secundária]
**Imagem principal:** [produto|pessoa|abstrato|lifestyle]
**Mensagem central:** [1 frase]
**Referências:** [descrever ou listar]
```

### 2. Variações obrigatórias de KV
- A: Versão principal (máxima qualidade)
- B: Versão alternativa de cor ou composição
- Dark: Versão dark mode
- Clean: Versão sem texto (para adaptações)

### 3. Derivações por formato
A partir do KV aprovado, derivar:
- Feed 1:1 (1080x1080)
- Feed 4:5 (1080x1350)
- Story 9:16 (1080x1920)
- Carrossel slide template
- Thumbnail de Reel

## Princípios de KV social

- **Legível em thumbnail**: Funciona a 1/10 do tamanho
- **Brand first**: Marca reconhecível em < 2 segundos
- **Flexible**: Adapta-se a qualquer formato sem perder impacto
- **Consistent**: Todas as peças derivadas são reconhecíveis como pertencendo à mesma campanha
