# Social Copywriting — Legendas e Copy

## Estrutura de legenda (Instagram/Facebook)

```
[HOOK — linha 1, para-scroll]

[DESENVOLVIMENTO — 3-5 linhas]
Explicar, educar, entreter ou inspirar.
Uma ideia por linha para respiração visual.

[CTA — acção específica]

.
.
.
[HASHTAGS — 10-15, mix de nicho e tendência]
```

## Hooks que param o scroll

### Tipos de hook
- **Pergunta:** "Já alguma vez te aconteceu X?"
- **Afirmação controversa:** "X está a matar o teu Y"
- **Número:** "3 erros que custaram X mil euros a Y"
- **Secreto:** "O que ninguém te diz sobre X"
- **Antes/depois:** "Antes: X. Depois: Y. A diferença?"
- **Comparação inesperada:** "X é como Y — e funciona da mesma forma"

### Regras do hook
- Máximo 10 palavras
- Nunca começar com o nome da marca
- Criar curiosidade sem resolver (cliffhanger)
- Falar directamente com o problema/desejo do público

## CTAs sociais eficazes

| Objectivo | CTA |
|---|---|
| Salvar conteúdo | "Guarda este post para quando precisares" |
| Comentários | "Conta-nos: qual é a tua experiência?" |
| Partilha | "Marca alguém que precisa ver isto" |
| Perfil | "Segue para mais dicas sobre X" |
| Link bio | "Link na bio para saber mais" |
| DM | "Envia-nos DM com a palavra X" |

## Tom por plataforma

| Plataforma | Tom | Comprimento |
|---|---|---|
| Instagram | Conversacional, aspiracional | 150-300 chars (preview) |
| TikTok | Casual, autêntico, trending | Roteiro falado, não texto |
| LinkedIn | Profissional, insights | 1.000-1.300 chars |
| Facebook | Comunitário, storytelling | 80-500 chars |
| Twitter/X | Directo, opinativo | < 280 chars |
