# Sites Copy Editing — Checklist de Qualidade

## Checklist obrigatório antes de entregar copy

### Clareza
- [ ] Proposta de valor compreensível em 5 segundos
- [ ] Sem jargão não explicado
- [ ] Frases com menos de 20 palavras (maioria)
- [ ] Um pensamento por frase

### Persuasão
- [ ] Headline usa uma das fórmulas testadas
- [ ] Benefits explícitos (não apenas features)
- [ ] CTA descreve resultado, não acção
- [ ] Objecções principais antecipadas

### Consistência
- [ ] Tom de voz uniforme em toda a página
- [ ] Terminologia consistente
- [ ] Pessoa gramatical consistente (2ª pessoa "você")

### Técnico (PT)
- [ ] Sem erros ortográficos
- [ ] Acentuação correcta
- [ ] Sem anglicismos desnecessários
- [ ] Pontuação correcta

## Erros comuns a eliminar
- "Somos líderes de mercado" → sem prova = vazio
- "Alta qualidade" → específico como?
- "Solução inovadora" → clichê total
- "Estamos aqui para si" → não diz nada
- Começar com o nome da empresa → começar com o cliente
