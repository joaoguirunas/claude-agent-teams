# Social Cinematic Composition — Linguagem Visual

## Regras de composição fundamentais

### Regra dos terços
```
Dividir o frame em 9 partes iguais (3x3).
Sujeito principal nos pontos de intersecção.
Horizonte numa das linhas horizontais.
```

### Linhas guia (Leading lines)
- Estradas, corredores, rios → guiam o olhar para o sujeito
- Usar para criar profundidade e direcção visual

### Enquadramento natural (Natural framing)
- Arcos, janelas, vegetação → enquadram o sujeito naturalmente
- Cria camadas e profundidade

### Simetria e padrões
- Funciona bem para arquitectura, flat lay, product shots
- Quebrar a simetria num ponto cria interesse

## Cor grading por mood

| Mood | Paleta | Implementação |
|---|---|---|
| Warm/Cozy | Laranja, amarelo, castanho | Highlights warm, shadows cool |
| Cool/Premium | Azul, cinzento, branco | Highlights cool, shadows deep |
| Moody/Dramatic | Tons escuros, contraste alto | Shadows crushed, highlights controlled |
| Airy/Clean | Brancos, bege, minimal | Low contrast, lifted shadows |
| Vintage | Dessaturado, fade, grain | Faded blacks, warm tones |

## Profundidade de campo (Bokeh)

**Quando usar:**
- Retratos (sujeito em foco, fundo desfocado)
- Product shots (produto em foco)
- Criar separação entre sujeito e fundo

**No prompt Freepik:**
```
85mm portrait lens, f/1.8 bokeh, shallow depth of field, 
subject sharp, background beautifully blurred
```

## Zona segura para texto (social)

```
Story/Reel 9:16:
  - Topo: 15% do frame → evitar (cortado em alguns devices)
  - Fundo: 20% do frame → evitar (interface da plataforma)
  - Zona segura: 65% central
  
Feed 1:1:
  - Bordas 5% → evitar
  - Zona segura: 90% central
```

## Paletas de cor por sector

| Sector | Paleta principal |
|---|---|
| Saúde/Wellness | Verde sage, branco, bege nude |
| Tecnologia | Azul profundo, branco, accent neon |
| Moda | Neutros + um accent bold |
| Alimentação | Quentes naturais, verde fresco |
| Finanças | Azul navy, dourado, branco |
| Criativo/Arte | Cores ousadas, contraste |
