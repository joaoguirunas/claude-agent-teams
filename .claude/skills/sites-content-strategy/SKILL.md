# Sites Content Strategy

## Hierarquia de mensagens

```
Nível 1 — Core Message (1 frase)
    "Fazemos X para Y obterem Z"

Nível 2 — Pilares de suporte (3-5 pontos)
    Cada pilar suporta o core message

Nível 3 — Provas (por pilar)
    Dados, testimonials, exemplos
```

## Arquitectura de conteúdo por página

### Home
- Proposta de valor principal
- Para quem é (audiência)
- Como funciona
- Prova social
- CTA para próxima acção

### About
- Origem e missão (StoryBrand: herói + guia)
- Valores (3-5, com descrição)
- Equipa (opcional)
- CTA de contacto ou produto

### Produto/Serviço
- Problema que resolve
- Como resolve (features → benefits)
- Casos de uso
- Preços (se aplicável)
- FAQ
- CTA de compra ou contacto

## Tom de voz por sector
- **Saúde:** Empático, claro, sem jargão
- **Tech/SaaS:** Directo, orientado a resultados
- **Moda/Lifestyle:** Aspiracional, sensorial
- **B2B:** Profissional, orientado a ROI
- **Educação:** Encorajador, claro, acessível
